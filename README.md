1. Choose a difficulty mode: **Easy / Medium / Hard**
2. Numbers start falling from the top.
3. You have **10 seconds** to:

   * Observe all falling numbers
   * Calculate their **total sum**
   * Enter the answer in the input box at the bottom

5. Game displays **Correct** or **Wrong** along with the actual sum.
